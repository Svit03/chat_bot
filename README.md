# Неруд Консультант — ML чат-бот для доставки нерудных материалов

## Актуальность проблемы

На торговых площадках и по телефону регулярно поступают однотипные вопросы от клиентов:
- "Сколько стоит щебень?"
- "Есть ли доставка?"
- "Какие материалы в наличии?"
- "Сколько будет стоить 3 тонны гравия с доставкой в Комушку?"

Ручной расчёт стоимости с учётом зоны доставки и типа материала занимает дополнительное время и повышает риск ошибок.

## Постановка задачи

Целью данной работы является разработка программного комплекса для автоматизации ответов на частые вопросы клиентов в сфере доставки нерудных материалов.

Для достижения поставленной цели необходимо решить следующие задачи:

1. Разработать ML-модель для классификации намерений пользователя по текстовым сообщениям.
2. Реализовать систему распознавания материалов и количества (цифры и словесное указание).
3. Разработать алгоритм расчёта стоимости материалов с учётом:
   - типа материала (сыпучий в тоннах или в мешках);
   - количества (от 1 до 4 тонн или от 1 до 100 мешков);
   - зоны доставки (район и микрорайон);
   - возможности бесплатной доставки для выбранных микрорайонов.
4. Создать клиент-серверную архитектуру с веб-интерфейсом чата.
5. Разработать административную панель для гибкого управления:
   - ценами на материалы;
   - зонами доставки и их стоимостью;
   - микрорайонами;
   - списком микрорайонов с бесплатной доставкой.
6. Обеспечить сохранение всех данных в реляционной базе данных PostgreSQL.
7. Реализовать возможность масштабирования и адаптации для других видов товаров и услуг.

## Решение

Был разработан чат-бот с ML-моделью, который автоматически отвечает на частые вопросы клиентов, рассчитывает стоимость материалов с доставкой и помогает с оформлением заказа.

В процессе разработки были изучены принципы классификации текста с помощью машинного обучения, реализована клиент-серверная архитектура на FastAPI и JavaScript, настроена интеграция с PostgreSQL и добавлена админ-панель для гибкого управления контентом.

## Архитектура и принцип работы

### Клиент-серверное взаимодействие

Бот разделён на две части:
- Клиентская часть (браузер) отвечает за отображение чата, отправку сообщений пользователя и визуализацию ответов. Поддерживает тёмную и светлую тему.
- Серверная часть (FastAPI) содержит ML-модель, обрабатывает запросы, определяет намерения пользователя и рассчитывает стоимость материалов с доставкой.

### Алгоритм классификации намерений

Модель обучена определять 5 типов намерений:
- price — запрос цены на материалы
- delivery — вопросы о доставке
- availability — наличие материалов
- contact — контакты компании
- greeting — приветствие

Параметры ML модели:
- Векторизация: TF-IDF на уровне символов (n-граммы 2-5)
- Классификатор: Логистическая регрессия (C=1.0)
- Размер выборки: 287 примеров
- Точность на кросс-валидации: 86.73%

### Система распознавания материалов

Бот понимает запросы с указанием количества:
- Цифры (2, 3, 4 тонны)
- Словесное указание (две тонны, три мешка)
- Обработка опечаток (грави -> гравий, щебнь -> щебень)

### Расчёт стоимости

Алгоритм учитывает:
1. Тип материала (сыпучий в тоннах или мешковой)
2. Количество (1-4 тонны или 1-100 мешков)
3. Зону доставки (4 района + микрорайоны)
4. Бесплатную доставку для выбранных микрорайонов

## Особенности архитектуры

### Управление зонами доставки

- Иерархическая структура: район -> микрорайоны
- У каждого района своя цена доставки сыпучих материалов
- Для мешковых материалов отдельная цена доставки
- Возможность настройки бесплатной доставки для конкретных микрорайонов

### База данных

Основные таблицы:
- materials — материалы и цены (тип: тонна или мешок)
- delivery_zones — районы доставки и их стоимость
- microdistricts — микрорайоны с привязкой к районам
- free_dolomite_microdistricts — микрорайоны с бесплатной доставкой

### Админ-панель

- Управление ценами на материалы
- Управление зонами доставки
- Управление микрорайонами
- Управление списком бесплатной доставки
- Базовая аутентификация (HTTP Basic Auth)

## Установка и запуск

### Требования

- Python 3.11 или выше
- PostgreSQL 15+
- Современный веб-браузер

### Инструкция по установке

1. Клонировать репозиторий

git clone https://github.com/Svit03/chat_bot.git

cd chat_bot

2. Создать виртуальное окружение

python -m venv venv

3. Активировать виртуальное окружение

Windows:
venv\Scripts\activate

macOS / Linux:
source venv/bin/activate

4. Установить зависимости

pip install -r requirements.txt

5. Настроить базу данных PostgreSQL

CREATE DATABASE nerud_bot;
-- Импортировать дамп из папки migrations/

6. Настроить переменные окружения

Создать файл .env:
DATABASE_URL=postgresql://postgres:password@localhost:5434/nerud_bot
ADMIN_USERNAME=admin
ADMIN_PASSWORD=admin123

7. Запустить бэкенд

python backend/app.py

8. Запустить фронтенд (отдельный терминал)

start frontend/index.html

start frontend/admin.html

9. Открыть в браузере

http://localhost:8001

## Управление

### Интерфейс чата

- Отправка сообщений через поле ввода или клавишу Enter
- Кнопки-подсказки для быстрых запросов
- Переключение тёмной/светлой темы
- Отображение уверенности ML-модели в ответе

### Примеры запросов

- \сколько стоит щебень\ — цена материала
- \2 тонны гравия\ — сразу спросит район
- \10 мешков доломита\ — расчёт с доставкой
- \доставка по улан удэ\ — информация о доставке
- \ваш телефон\ — контактные данные

### Админ-панель

Доступ: http://localhost:8001/admin.html

- Логин: admin
- Пароль: admin123

Возможности:
- Редактирование цен материалов (тонна и мешок)
- Управление зонами доставки
- Управление микрорайонами
- Настройка бесплатной доставки доломита

## ML модель

Метрики обученной модели:

| Метрика | Значение |
|---------|----------|
| Размер обучающей выборки | 287 примеров |
| Количество классов | 5 интентов |
| Точность (CV-3) | 86.73% |
| Метод векторизации | TF-IDF (char_wb, ngram 2-5) |
| Классификатор | LogisticRegression (C=1.0) |

Распознавание материалов:

- Исправление опечаток (нормализация текста)
- Нечёткий поиск материалов (fuzzy matching)
- Контекстное сохранение состояния диалога
